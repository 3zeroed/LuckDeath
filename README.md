## 一个用来增加游戏趣味性的小插件

> 玩家死亡后会随机掉落背包内物品
> 
> 配置文件可以设置物品掉落几率
> 
> 另外本插件自带死亡不掉落
> 
> 想要几率性掉落物品的效果
> 
> 就必须开启死亡不掉落
>
